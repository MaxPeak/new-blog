# 你想做明星吗

### Sentence Structure（句子结构）

- Do you want to be a celebrity?
  - Do you want to be a celebrity?
  - 译：你想做明星吗?

  - 译：也许有一天我会出名，但那对我来说不是很重要。你可以成功而不出名，也许不出名更好，因为至少你有隐私。
